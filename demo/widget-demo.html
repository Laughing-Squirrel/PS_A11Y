Accessibility Widget Demo

Click the accessibility button on the right side to open the A11Y control panel

Information: This page demonstrates the A11Y Widget integrated with PeopleSoft FSCM 9.2 Classic Plus interface.
About This Demo

This is a standalone demonstration of the A11Y Widget integrated with PeopleSoft FSCM 9.2 Classic Plus styling. The widget provides comprehensive accessibility controls that allow users to customize their browsing experience according to their individual needs.

Try the different features using the widget panel, or use the keyboard shortcuts listed below.

Keyboard Shortcuts

Use these keyboard shortcuts for quick access to accessibility features:

Alt + A Toggle panel
Alt + + Increase font
Alt + - Decrease font
Alt + 0 Reset font
Alt + C Cycle contrast
Alt + R Reading guide
Esc Close panel
Contrast Mode Testing

Use the contrast mode buttons in the widget to switch between:

  • Normal: Default PeopleSoft colors
  • Dark: Dark background with light text
  • Light: High contrast light mode
  • Invert: Inverts all colors on the page
